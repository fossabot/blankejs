# Controller

	pad.1 ... pad.16

## GameCube Adapter

**Pad numbers**

	1 - Y

	2 - A 

	3 - B 

	4 - X

	5 - L

	6 - R

	8 - Z

	10 - Start

	13 - D Up

	14 - D Right

	15 - D Down

	16 - D Left

**Axis numbers**

	1 - L Stick X

	2 - L Stick Y

	3 - C Stick Y

	4 - L Trigger pressure (up = negative, down = positive, 0 = half way)

	5 - R Trigger pressure

	6 - C Stick X
